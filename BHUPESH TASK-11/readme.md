🖥️ Opening HTML Files (to view in browser)
Method 1 — Double Click (Easiest)
Just double-click your index.html file in your file explorer. It will open directly in your default browser.
Method 2 — Drag & Drop
Drag the .html file and drop it into any open browser window.
Method 3 — From Browser
Open your browser → press Ctrl + O (Windows) or Cmd + O (Mac) → navigate to your file → click Open.
